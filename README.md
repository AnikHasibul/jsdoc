# jsdoc
A [godoc](https://godoc.org) like cli tool for nodejs.

# Installation

```sh
go get -v github.com/AnikHasibul/jsdoc
```

# Usage:

```txt
$ jsdoc

USAGE of jsdoc:
 jsdoc <module>
 jsdoc <sym>[.<method>]                  
 jsdoc [<module>].<sym>[.<method>]

EXAMPLE USAGE of jsdoc:
 jsdoc http
 jsdoc http.Agent

CREATED BY:
 @AnikHasibul

```
